# ARAS ![AR-Pancreas-Teaser](https://github.com/user-attachments/assets/468aba15-fd45-4689-a0dc-5ba505c7a303)

ARAS is wearable AR assistance system designed for intraoperative navigation in pancreatic surgery. ARAS was developed following a user-centered design process advised by experienced surgeons and was iteratively refined and evaluated in clinical trials.
